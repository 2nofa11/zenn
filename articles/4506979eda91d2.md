### 挿入ソートアルゴリズムの再現

* 経験のあるプログラマーの短期記憶部分が抜け落ちていて、再現することができなかった。
  コメントを入れてビーコンのするのは確かに大事だな。

  ```ts
  let arrayList = [45,21,12,14,8,1]
  let temp
  
  for(let i = 0;i < arrayList.length ;i++){
      for(let j = i; 0 < j;j--){
          // ここを再現できなかった。
          if(arrayList[j]<arrayList[j-1]){
              temp = arrayList[j]
              arrayList[j] = arrayList[j -1]
              arrayList[j -1] = temp
          }
      }
  }
  console.log(arrayList)
  ```
